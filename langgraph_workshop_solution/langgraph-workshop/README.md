# 🍽️ 餐廳推薦 Agent — Vibe Coding 工作坊

## 這是什麼？
工作坊的 starter template。用 Antigravity 的 AI Agent 把它變成完整的應用。

## 資料夾裡有什麼？

```
langgraph-workshop/
├── data/
│   └── restaurants.py  ← 餐廳資料庫（Agent 會用到）
├── .env.example        ← API Key 範本
├── requirements.txt    ← Python 依賴
└── docs/
    └── student-handout.md  ← 學員講義
```

## 開始

1. 用 Antigravity 開啟這個資料夾
2. 翻開 `docs/student-handout.md`
3. 跟著講師的節奏走！
